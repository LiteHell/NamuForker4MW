나무위키 문서를 미디어위키 XML로 포크하는 프로그램입니다. 한번에 두개 이상의 문서를 포크할 순 없습니다.
라이선스 : GPL-3.0

# 필요한 프로그램들
* node.js **최신버전**

# 설치 방법
```
npm install
npm install-dev
```
아직 디버그용 코드들을 정리하지 못했습니다.

# 사용 방법
```
node index.js
```

# 주의사항
* 아직 미디어위키에 안 올려봐서 버그가 있는지 없는지 모릅니다.
* 리버전 갯수랑 필요한 시간이 비례합니다.
* 발코딩입니다. 제가 야매로 배운 사람이라서 그렇습니다.
